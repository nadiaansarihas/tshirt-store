# NEON THREADS - Y2K Streetwear Website

## Overview
A vibrant Y2K-themed e-commerce website showcasing streetwear products with neon aesthetics, interactive animations, and a retro-futuristic design.

## Project Type
Static website (HTML, CSS, JavaScript)

## Tech Stack
- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6)
- **Fonts**: Google Fonts (Orbitron, Press Start 2P)
- **Assets**: Static images in `/static/images/`

## Project Structure
```
.
├── index.html          # Main HTML file
├── styles.css          # All CSS styling
├── script.js           # JavaScript functionality
└── static/
    └── images/         # Product and background images
```

## Features
- Product catalog with filtering (All, Graphic Tees, Hoodies, Accessories)
- Shopping cart functionality
- Animated UI elements (glitch effects, cursor trail, etc.)
- Interactive terminal with vibe check
- Statistics counter animations
- Easter eggs (Konami code)
- Fully responsive design

## Setup Notes
- No build process required
- No dependencies to install
- Runs on any static file server
- Port 5000 for Replit webview
