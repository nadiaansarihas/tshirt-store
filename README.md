# NEON THREADS - Y2K Streetwear E-Commerce Website

A vibrant Y2K-themed e-commerce website for selling streetwear, featuring neon aesthetics, interactive animations, and a retro-futuristic design. Built as part of the Vibe Coding 101 course.

## Features

- Product catalog with 12+ streetwear items (graphic tees, hoodies, accessories)
- Shopping cart functionality with add/remove items
- Product filtering by category (All, Graphic Tees, Hoodies, Accessories)
- Responsive design for mobile and desktop
- Interactive UI elements (glitch effects, custom cursor trail, animated stats)
- Interactive terminal with "vibe check" feature
- Trending product badges
- Easter egg: Konami code for secret discount

## Technologies Used

- HTML5 for structure
- CSS3 for styling, animations, and responsive design
- JavaScript (ES6) for interactivity
- Google Fonts (Orbitron, Press Start 2P)
- Replit for development environment
